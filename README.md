# 🏔️ Trek Maharashtra

A simple and responsive website that showcases **popular trekking destinations in Maharashtra**.
This project is built using **HTML, CSS, and JavaScript** and provides information about different trekking spots for adventure lovers.

---

## 🌐 Live Website

🔗 https://pranalishinde06.github.io/trek-maharashtra-html-css-js/

## 📌 Features

* 🏔️ Information about famous trekking places in Maharashtra
* 📱 Fully responsive website
* 🖼️ Beautiful images of trekking locations
* 🧭 Simple and easy navigation
* ⚡ Fast static website using HTML, CSS, JS
